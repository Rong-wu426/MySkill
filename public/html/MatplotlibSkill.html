Matplotlib 筆記

簡介

Matplotlib 是一個 Python的資料可視化庫，可以用來產生各種靜態、動態和互動式圖表。

安裝


# 使用 pip 安裝 Matplotlib
pip install matplotlib

折線圖

使用plt.plot() 方法來創建折線圖


import matplotlib.pyplot as plt

# 繪製基本圖形
plt.plot([1, 2, 3, 4], [10, 20, 25, 30])
plt.xlabel('X 軸') # 添加x標籤
plt.ylabel('Y 軸') # 添加y標籤
plt.title('基本圖形') # 添加標題
plt.legend()  # 添加圖例
plt.show()   # 顯示圖表

散點圖

使用plt.scatter() 方法來創建散點圖


import matplotlib.pyplot as plt

# 繪製散點圖
x = [1, 2, 3, 4]
y = [10, 20, 15, 25]

plt.scatter(x, y, color='red', marker='x')
plt.xlabel('X 軸')
plt.ylabel('Y 軸')
plt.title('散點圖')
plt.show()

自定義點的大小和顏色


sizes = [20, 50, 80, 200, 500, 100, 300, 400, 700, 900]
colors = np.random.rand(10)  # 隨機顏色

直方圖

使用 plt.hist() 來創建直方圖


import matplotlib.pyplot as plt

# 繪製直方圖
data = [1, 2, 2, 3, 3, 3, 4, 4, 4, 4]

plt.hist(data, bins=4, color='blue', edgecolor='black')
plt.xlabel('數值')
plt.ylabel('頻次')
plt.title('直方圖')
plt.show()

水平柱狀圖

創建一個水平的柱狀圖，可以使用 plt.barh()：


# 繪製水平柱狀圖
plt.barh(categories, values, color='skyblue')

plt.title('Horizontal Bar Chart')
plt.xlabel('Values')
plt.ylabel('Categories')

plt.show()

圓餅圖

使用 plt.pie() 可以繪製餅圖


import matplotlib.pyplot as plt

# 準備數據
labels = ['A', 'B', 'C', 'D']
sizes = [15, 30, 45, 10]
colors = ['gold', 'yellowgreen', 'lightcoral', 'lightskyblue']

plt.pie(sizes, labels=labels, colors=colors, autopct='%1.1f%%', startangle=140)
# autopct='%1.1f%%' 會在每個區域內顯示百分比
# startangle=140 表示從 140 度開始繪製餅圖

# 確保餅圖是圓形的
plt.axis('equal')

# 顯示圖表
plt.title('Pie Chart Example')
plt.show()

自定義


import matplotlib.pyplot as plt

# 繪製自訂圖形
x = [1, 2, 3, 4]
y = [10, 20, 15, 25]

plt.plot(x, y, color='green', linestyle='--', marker='o', markersize=8)
plt.xlabel('X 軸')
plt.ylabel('Y 軸')
plt.title('自定義圖形')
plt.grid(True, linestyle='-.')
plt.show()

調整圖表風格

使用 plt.style.use() 來應用不同的風格：


plt.style.use('ggplot')  # 使用 ggplot 風格

# 繪製示例圖表
plt.plot(x, y)
plt.title('Styled Plot')
plt.show()

常見的風格包括 ggplot、seaborn、bmh、dark_background 等

子圖

使用 plt.subplots() 來創建多個子圖：


import matplotlib.pyplot as plt

# 建立2行2列的子圖
fig, axs = plt.subplots(2, 2, figsize=(10, 8)) # figsize 用於設置圖的尺寸

# 第一個子圖
axs[0, 0].plot([1, 2, 3], [1, 4, 9], 'tab:blue') # 折線圖
axs[0, 0].set_title('子圖 1')

# 第二個子圖
axs[0, 1].scatter([1, 2, 3], [9, 4, 1], color='red') # 散點圖
axs[0, 1].set_title('子圖 2')

# 第三個子圖
axs[1, 0].bar(['A', 'B', 'C'], [10, 20, 15], color='green') # 柱狀圖
axs[1, 0].set_title('子圖 3')

# 第四個子圖
axs[1, 1].hist([1, 2, 2, 3, 3, 4], bins=3, color='purple') # 直方圖
axs[1, 1].set_title('子圖 4')

# 使用 tight_layout()會自動調整子圖的布局
plt.tight_layout() 
plt.show()
