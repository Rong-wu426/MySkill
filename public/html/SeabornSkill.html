簡介

Seaborn 是一個基於 Matplotlib 的 Python 資料視覺化庫，它簡化了繪製複雜統計圖表的過程，適合進行數據分析和探索性資料分析 (EDA)。Seaborn 提供了更高階的 API，讓繪製統計圖表變得簡單直觀。這些圖表包括散點圖、箱型圖、熱力圖等，且自動處理數據集中分類變量的分組與著色。

安裝

# 使用 pip 安裝 Seaborn
pip install seaborn

這將會自動安裝 Seaborn 以及 Matplotlib 等相關依賴。你可以使用 import seaborn as sns 來加載 Seaborn 並開始使用它。

基本繪圖

Seaborn 的基本繪圖功能可以讓我們快速創建統計圖表。以下是使用 Seaborn 繪製散點圖的範例，展示兩個連續變量之間的關係。在這個例子中，我們使用了 tips 數據集，該數據集包含了顧客的消費與小費信息。

import seaborn as sns
import matplotlib.pyplot as plt

# 加載示例數據集
data = sns.load_dataset("tips")

# 繪製散點圖
sns.scatterplot(x="total_bill", y="tip", data=data)
plt.xlabel('總帳單')
plt.ylabel('小費')
plt.title('總帳單 vs 小費')
plt.show()

這個範例展示了如何繪製「總帳單」與「小費」之間的關係圖。Seaborn 自動為你處理了許多細節，如標籤、顏色等。

直方圖

直方圖是一種常用的統計圖表，用於顯示數據的分佈情況。Seaborn 的 histplot 函數使得繪製直方圖變得非常簡單。以下範例使用 Iris 數據集，展示了不同種類花萼長度的分佈。

import seaborn as sns
import matplotlib.pyplot as plt

# 加載示例數據集
data = sns.load_dataset("iris")

# 繪製直方圖，按照物種分類顯示
sns.histplot(data=data, x="sepal_length", hue="species", multiple="stack")
plt.xlabel('花萼長度')
plt.ylabel('頻次')
plt.title('花萼長度的直方圖')
plt.show()

這裡使用了 hue 參數來根據「species」列進行著色分組，並使用 multiple="stack" 參數來將不同物種的分佈堆疊起來。

散點圖

散點圖是用來顯示兩個變量之間的關係。在 Seaborn 中，scatterplot 函數可以非常方便地繪製出散點圖。以下範例展示了花萼長度與花萼寬度之間的關係，並根據物種進行著色。

import seaborn as sns
import matplotlib.pyplot as plt

# 加載示例數據集
data = sns.load_dataset("iris")

# 繪製散點圖
sns.scatterplot(x="sepal_length", y="sepal_width", hue="species", data=data)
plt.xlabel('花萼長度')
plt.ylabel('花萼寬度')
plt.title('花萼長度 vs 花萼寬度')
plt.show()

我們使用 hue 參數來根據物種進行顏色區分，讓我們能清楚地看到不同物種之間的差異。

箱型圖

箱型圖顯示數據的分佈情況，包括數據的四分位範圍、中位數以及異常值。使用 Seaborn 的 boxplot 函數可以輕鬆繪製箱型圖。以下範例展示了根據星期劃分的消費數據。

import seaborn as sns
import matplotlib.pyplot as plt

# 加載示例數據集
data = sns.load_dataset("tips")

# 繪製箱型圖
sns.boxplot(x="day", y="total_bill", data=data)
plt.xlabel('星期')
plt.ylabel('總帳單')
plt.title('各星期的總帳單箱型圖')
plt.show()

箱型圖展示了不同星期的總帳單數據分佈。你可以清楚地看到數據的異常值以及數據的範圍。

對角線圖

對角線圖是 Seaborn 提供的一種強大的工具，用於顯示數據集中每對變量之間的關係。pairplot 函數可以自動繪製數據集中所有變量的成對關係圖。

import seaborn as sns
import matplotlib.pyplot as plt

# 加載示例數據集
data = sns.load_dataset("iris")

# 繪製對角線圖
sns.pairplot(data, hue="species")
plt.title('對角線圖')
plt.show()

這個圖展示了 Iris 數據集中每個變量之間的關係，並根據不同物種進行著色。它可以快速幫助我們發現變量之間的潛在關聯。

熱力圖

熱力圖是一種顏色編碼的矩陣圖，用來展示兩個變量之間的關係。Seaborn 的 heatmap 函數可以幫助我們繪製熱力圖，並提供多種配色選項。以下範例展示了一個隨機數據集的熱力圖。

import seaborn as sns
import matplotlib.pyplot as plt
import numpy as np

# 創建隨機數據集
data = np.random.rand(10, 12)

# 繪製熱力圖
sns.heatmap(data, cmap="YlGnBu", annot=True)
plt.title('熱力圖')
plt.show()

這個熱力圖使用了 YlGnBu 配色方案，並在每個格子中顯示數值。annot=True 參數用來在每個單元格中標註數值。

自定義

Seaborn 提供了豐富的自定義選項來修改圖形的外觀。你可以修改顏色、標記樣式、圖形大小等。以下範例展示了如何自定義散點圖。

import seaborn as sns
import matplotlib.pyplot as plt

# 加載示例數據集
data = sns.load_dataset("tips")

# 自定義散點圖的顏色和標記
sns.scatterplot(x="total_bill", y="tip", data=data, color="red", marker="x")
plt.xlabel('總帳單')
plt.ylabel('小費')
plt.title('自定義散點圖')
plt.show()

散點圖的顏色設置為紅色，標記設置為「x」符號，並且可以隨意修改圖表標題、軸標籤和圖形樣式。
