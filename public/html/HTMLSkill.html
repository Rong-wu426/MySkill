HTML 筆記

DOCTYPE

<!DOCTYPE html>

DOCTYPE 用於告訴瀏覽器當前文檔的 HTML 版本，通常在 HTML 文檔的開頭。

標題


    <h1>主標題</h1>
    <h2>副標題</h2>
    <h3>次要標題</h3>
    <h4>更小的標題</h4>
    <h5>更小的標題</h5>
    <h6>最小的標題</h6>

HTML 提供了六個層級的標題元素，從 <h1> 到 <h6>，其中 <h1> 代表最重要的標題，<h6> 代表最不重要的標題。

段落

<p>這是一段文本</p>

<p> 元素用於定義段落。

換行

<br>

<br> 元素用於在文本中插入換行。

連結

<a href="https://example.com">這是一個連結</a>

<a> 元素用於創建超連結。屬性 href 指定了連結的目標 URL。

圖片

<img src="image.jpg" alt="描述">

<img> 元素用於顯示圖片。 屬性 src 指定了圖片的 URL， 屬性 alt 提供了圖片的描述。

文字格式


    <b>粗體文本</b>
    <strong>重要文本</strong>
    <i>斜體文本</i>
    <em>強調文本</em>
    <mark>標記文本</mark>
    <small>小字文本</small>
    <del>刪除文本</del>
    <ins>插入文本</ins>
    <sub>下標文本</sub>
    <sup>上標文本</sup>

這些元素用於格式化文本，以增強其可讀性或提供額外的語意：

  • <b>: 粗體文本。
  • <strong>: 重要文本。
  • <i>: 斜體文本。
  • <em>: 強調文本。
  • <mark>: 標記文本。
  • <small>: 小字文本。
  • <del>: 刪除文本。
  • <ins>: 插入文本。
  • <sub>: 下標文本。
  • <sup>: 上標文本。

其他元素

表格：


    <table>
    <tr>
    <th>表頭</th>
    <th>表頭</th>
    </tr>
    <tr>
    <td>數據</td>
    <td>數據</td>
    </tr>
    </table>

列表：


    <ul>
    <li>項目 1</li>
    <li>項目 2</li>
    </ul>
  

    <ol>
    <li>項目 1</li>
    <li>項目 2</li>
    </ol>

區塊和行內元素：


    <div>區塊元素</div>
    <span>行內元素</span>

類別和 ID：


    <div class="container">...</div>
    <div id="unique">...</div>

類別（class）用於定義樣式，ID 用於唯一標識。
