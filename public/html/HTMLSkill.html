HTML 筆記

環境創建


# 創建資料夾
mkdir my-website

# 進入資料夾
cd my-website

# 創建 index.html 文件
touch index.html

# 編輯 index.html 文件，輸入基本 HTML 結構
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>我的網站</title>
</head>
<body>
  <h1>歡迎來到我的網站</h1>
</body>
</html>

DOCTYPE


<!DOCTYPE html>

DOCTYPE 用於告訴瀏覽器當前文檔的 HTML 版本，通常在 HTML 文檔的開頭。

meta 標籤


<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="description" content="網站描述">
  • <meta> 標籤用於提供有關 HTML 文檔的元數據。
  • charset: 指定文檔的字符編碼。
  • viewport: 設定瀏覽器視口，控制頁面縮放。
  • description: 提供網頁描述，用於 SEO。

註解


<!-- 這是一個註解 -->

註解用於在 HTML 文檔中添加說明性文字，不會顯示在頁面上。

標題


<h1>主標題</h1>
<h2>副標題</h2>
<h3>次要標題</h3>
<h4>更小的標題</h4>
<h5>更小的標題</h5>
<h6>最小的標題</h6>

HTML 提供了六個層級的標題元素。

從 <h1> 到 <h6>， 其中<h1> 代表最重要的標題，<h6> 代表最不重要的標題。

段落


<p>這是一段文本</p>

<p> 元素用於定義段落。

換行


<br>

<br> 元素用於在文本中插入換行。

連結


<a href="https://example.com">這是一個連結</a>

<a> 元素用於創建超連結。屬性 href 指定了連結的目標 URL。

圖片


<img src="image.jpg" alt="描述">

<img> 元素用於顯示圖片

  • 屬性 src 指定了圖片的 URL
  • 屬性 alt 提供了圖片的描述

影片

<video width="640" height="360" controls> <source src="movie.mp4" type="video/mp4"> 您的瀏覽器不支持 video 元素。 </video>

<video> 元素用於嵌入影片。可以使用以下屬性：

  • width 和 height: 設定影片的寬度和高度。
  • controls: 顯示播放控制器。

<source> 元素指定影片文件的來源和格式。如果瀏覽器不支持影片格式，則會顯示 source 元素中的文本。

文字格式


<b>粗體文本</b>
<strong>重要文本</strong>
<i>斜體文本</i>
<em>強調文本</em>
<mark>標記文本</mark>
<small>小字文本</small>
<del>刪除文本</del>
<ins>插入文本</ins>
<sub>下標文本</sub>
<sup>上標文本</sup>

這些元素用於格式化文本，以增強其可讀性或提供額外的語意：

  • <b>: 粗體文本。
  • <strong>: 重要文本。
  • <i>: 斜體文本。
  • <em>: 強調文本。
  • <mark>: 標記文本。
  • <small>: 小字文本。
  • <del>: 刪除文本。
  • <ins>: 插入文本。
  • <sub>: 下標文本。
  • <sup>: 上標文本。

輸入標籤


<input type="text" placeholder="輸入文本">
<textarea placeholder="輸入多行文本">
</textarea>
  • <input> 元素用於創建輸入字段。可以使用 type 屬性指定輸入類型。
  • <textarea> 元素用於創建多行文本框。

區塊和行內元素


<div>區塊元素</div>
<span>行內元素</span>

區塊元素（如 <div>）會在頁面中創建一個新的區塊，而行內元素（如 <span>）則不會打斷文本流。

類別和 ID


<div class="container">...</div>
<div id="unique">...</div>

類別（class）用於定義樣式，ID 用於唯一標識。

表格


<table>
  <tr>
    <th>表頭</th>
    <th>表頭</th>
  </tr>
  <tr>
    <td>數據</td>
    <td>數據</td>
  </tr>
</table>
  • <table> 元素用於創建表格
  • <tr> 定義行
  • <th> 定義表頭單元格
  • <td> 定義普通單元格

列表


<ul>
  <li>項目 1</li>
  <li>項目 2</li>
</ul>
<ol>
  <li>項目 1</li>
  <li>項目 2</li>
</ol>

列表元素用於顯示有序列表和無序列表。

- <ul>: 無序列表。

- <ol>: 有序列表。
