安裝

  1. 使用CDN引入:
    2. <script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
  2. 在本地使用:
    3. <script src="path/to/jquery.js"></script>

基本語法

$(選擇器).動作();
  • $:這是jQuery的符號，表示要使用jQuery
  • 選擇器：就像 CSS 選擇器一樣，用來選擇 HTML 元素
  • 動作：你想對選中的元素執行的操作或效果

選擇器

$(document).ready(function(){
  // 選擇所有的 <p> 元素
  $("p").css("color", "blue");

  // 選擇具有特定 ID 的元素
  $("#myId").css("background-color", "yellow");

  // 選擇具有特定 class 的元素
  $(".myClass").css("font-size", "20px");
});

DOM 操作

    DOM（Document Object Model，文檔對象模型）是一個用於表示結構化文檔的編程接口
  • append() 和 prepend()：在元素內部添加內容
    • // 動態添加一個列表項目
$("ul").append("<li>新項目</li>");
  • before() 和 after()：在元素外部添加內容
    • // 在元素前面插入一個段落
$("h2").before("<p>這是插入在標題前的段落</p>");
  • remove()：刪除元素
    • // 刪除所有的段落
$("p").remove();
  • html() 和 text()：獲取或設置 HTML 和文字內容
    • // 獲取或設置 HTML 內容
$("div").html("<p>這是新的 HTML 內容</p>");
  • parent()：獲取當前元素的父元素
    • // 獲取父元素
$("li").parent().css("background-color", "lightgray");
  • children()：獲取當前元素的所有子元素
    • // 獲取子元素
$("ul").children().css("color", "blue");
  • find()：在當前元素內查找特定的子元素
    • // 查找子元素
$("div").find("p").css("font-weight", "bold");
  • siblings()：獲取當前元素的兄弟元素
    • // 獲取兄弟元素
$("li.active").siblings().hide();
  • next() 和 prev()：分別獲取當前元素的下一個或上一個兄弟元素
    • // 獲取下一個兄弟元素
$("h2").next().css("color", "red");

// 獲取上一個兄弟元素
$("h2").prev().css("color", "green");

改變 CSS 樣式

  • css() ：獲取或設置 CSS 樣式
  • addClass() 和 removeClass()：添加或移除 class
// 改變所有段落的顏色為藍色
$("p").css("color", "blue");

顯示/隱藏元素

  • hide()：隱藏元素
  • show()：顯示元素
  • toggle()：切換顯示狀態
// 點擊按鈕時切換段落顯示狀態
$("button").click(function() {
    $("p").toggle();
});

事件處理

  • 常見的事件處理方法：
    • .click() 和.dblclick() 用於處理用戶的點擊和雙擊行為
    • .mouseenter() 和.mouseleave() 用於處理滑鼠進入和離開元素的事件
    • .keydown() 和.keyup() 用於處理鍵盤按鍵的按下和釋放事件
    • .change() 用於處理表單元素值變更的事件
    // 當按鈕被點擊時，顯示一個警告框
$("button").click(function() {
  alert("按鈕被點擊了！");
});
  • on()：用於事件委派 
    // 事件委派：對所有未來加入的 .item 元素處理點擊事件
$("ul").on("click", ".item", function() {
  alert("列表項目被點擊！");
});
  • hover()：處理滑鼠進入和離開事件 
    // 滑鼠懸停事件：當鼠標懸停在 .box 元素上時，改變顏色
$(".box").hover(
  function() {
    $(this).css("background-color", "yellow");
  },
  function() {
    $(this).css("background-color", "white");
  }
);

效果

  • fadeIn() 和fadeOut() ：淡入淡出效果
    • $("button").click(function() {
  $("p").fadeIn("slow");
});
  • fadeTo()：指定透明度進行淡入淡出效果
    • $("button").click(function() {
  $("p").fadeTo("slow", 0.5);
});
  • slideUp() 和slideDown() ：滑動顯示和隱藏
    • $("button").click(function() {
  $("p").slideUp();
});
  • animate()：進行自定義動畫
    • $("button").click(function() {
  $("div").animate({ width: "300px" }, 1000);
});

AJAX

  • $.ajax():可以發送任何類型的 HTTP 請求（GET、POST、PUT、DELETE 等）
    • $.ajax({
      url: "server-endpoint",
      method: "GET", // 也可以是 POST、PUT、DELETE 等
      data: { key: "value" }, // 發送給伺服器的數據
      success: function(response) {
        console.log("成功接收到數據", response);
      },
      error: function(error) {
        console.log("發生錯誤", error);
      }
    });
  • $.get():通常用來從伺服器獲取數據
    • $.get("server-endpoint", { key: "value" }, function(response) {
      console.log("GET 請求成功", response);
    });
  • $.post():通常用來提交數據到伺服器
    • $.post("server-endpoint", { key: "value" }, function(response) {
      console.log("POST 請求成功", response);
    });
