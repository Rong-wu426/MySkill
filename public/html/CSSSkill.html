顏色

顏色可以通過多種方式在 CSS 中設置：

  • 顏色代碼：使用十六進位制代碼表示顏色，例如 #FF5733。
  • 顏色名稱：使用預定義的顏色名稱，例如 red。
  • RGB：通過紅、綠、藍三個原色的組合來表示顏色，例如 rgb(255, 87, 51)。
  • HSL：通過色調、飽和度和亮度來表示顏色，例如 hsl(9, 100%, 60%)。

/* 顏色代碼 */
.example-color-code {
  color: #FF5733; /* 橙色 */
}

/* 顏色名稱 */
.example-color-name {
  color: red; /* 紅色 */
}

/* RGB 三原色 */
.example-rgb {
  color: rgb(255, 87, 51); /* 橙色 */
}

/* HSL 顏色 */
.example-hsl {
  color: hsl(9, 100%, 60%); /* 橙色 */
}

盒子模型

盒子模型（Box Model）定義了元素的內容、內邊距、邊框和外邊距之間的關係：

  • padding：控制內容與邊框之間的內邊距。
  • margin：控制元素與外部其他元素之間的外邊距。
  • border：控制元素的邊框樣式、顏色和寬度。

.box {
  width: 100px;
  padding: 10px; /* 內部空間 */
  border: 1px solid black; /* 邊框 */
  margin: 20px; /* 外部空間 */
}

float & display

這裡是關於 float 和 display 屬性：

  • float：將元素向左或向右浮動，使其與周圍的文本和行內元素一起排列。
  • display：控制元素的顯示方式，如塊級元素（block）、行內元素（inline）或隱藏元素（none）。

/* Float */
.float-example {
  float: left; /* 向左浮動 */
}

/* Display */
.display-block {
  display: block; /* 塊級元素 */
}

.display-inline {
  display: inline; /* 內聯元素 */
}

.display-none {
  display: none; /* 隱藏元素 */
}

定位

定位屬性控制元素在頁面中的位置：

  • absolute：相對於最近的已定位祖先元素（非 static），精確定位元素。
  • relative：相對於元素自身的正常位置進行定位，保留元素在文檔流中的空間。
  • fixed：相對於瀏覽器窗口進行定位，滾動頁面時位置不變。
  • sticky：元素在特定滾動位置變得固定。

.positioned {
  position: absolute; /* 絕對定位 */
  top: 10px;
  left: 20px;
}

opacity & border-radius

opacity 和 border-radius 用來控制元素的透明度和圓角：

  • opacity：設置元素的透明度，取值範圍是 0 到 1，0 表示完全透明，1 表示不透明。
  • border-radius：設置元素的邊框圓角，值可以是像素、百分比或其他單位。

.opacity-example {
  opacity: 0.5; /* 透明度 */
}

.border-radius-example {
  border-radius: 10px; /* 圓角 */
}

類別和ID

使用 class 和 ID 來選擇和設置元素的樣式：

  • class：可以應用到多個元素，使用 . 開頭。
  • ID：唯一標識一個元素，使用 # 開頭。

/* Class */
.my-class {
  color: blue;
}

/* ID */
#my-id {
  color: red;
}

選擇器

CSS 提供多種選擇器來選擇元素並應用樣式：

  • 元素選擇器：選擇所有指定元素，例如 p 選擇所有段落。
  • 類別選擇器：選擇所有具有特定類別的元素，例如 .my-class。
  • ID 選擇器：選擇具有特定 ID 的元素，例如 #my-id。
  • 群組選擇器：將多個選擇器組合在一起，使用逗號分隔，例如 h1, h2, h3。
  • 後代選擇器：選擇某元素內部的所有指定元素，例如 div p。
  • 子元素選擇器：選擇某元素內部的直接子元素，例如 ul > li。

/* 元素選擇器 */
p {
  color: green;
}

/* 類別選擇器 */
.my-class {
  font-size: 16px;
}

/* ID 選擇器 */
#my-id {
  background-color: yellow;
}

/* 群組選擇器 */
h1, h2, h3 {
  font-weight: bold;
}

/* 後代選擇器 */
div p {
  margin: 0;
}

/* 子元素選擇器 */
ul > li {
  list-style-type: square;
}

動畫

使用 animation 和 transition 來創建動畫效果：

  • animation：創建複雜的動畫效果，通過定義關鍵幀來指定動畫的各個階段。
  • transition：在元素狀態發生變化時，創建過渡效果。

/* Animation */
@keyframes example {
  from { background-color: red; }
  to { background-color: yellow; }
}

.animation-example {
  animation: example 3s infinite;
}

/* Transition */
.transition-example {
  transition: background-color 0.5s ease;
}

.transition-example:hover {
  background-color: blue;
}
